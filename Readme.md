# 使い方
```bash
$ git clone git://github.com/tune/dotfiles.git
$ cd dotfiles
$ git submodule init
$ git submodule update
$ ./setup.sh
```

vimを起動する前にNeoBundleをインストールしておく。
```bash
git clone git://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
```

vimを起動したら`:BundleInstall`を実行する。
